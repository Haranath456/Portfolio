
# Haranath Portfolio Website

## Overview

This is a personal portfolio website showcasing the work and skills of Haranath, a passionate full-stack developer. The website features a modern, responsive design with interactive elements, video backgrounds, and smooth animations to provide an engaging user experience.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Interactive Elements**: Hover effects on project videos, animated skill sliders
- **Video Backgrounds**: Dynamic video elements throughout the site
- **Contact Form**: Integrated EmailJS for direct messaging
- **Social Links**: Direct links to GitHub, LinkedIn, and email
- **Multiple Pages**: Main portfolio, additional projects, and certifications sections
- **Smooth Animations**: AOS (Animate On Scroll) library for scroll-triggered animations

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Custom styling with flexbox, grid, and media queries
- **JavaScript**: Interactive functionality and DOM manipulation
- **Libraries**:
  - AOS (Animate On Scroll)
  - Boxicons for icons
  - EmailJS for contact form functionality

## How to View

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Navigate through the sections using the menu or scroll

## Sections

- **Hero**: Introduction with video background
- **About**: Personal information and tech stack
- **Projects**: Featured work with video previews
